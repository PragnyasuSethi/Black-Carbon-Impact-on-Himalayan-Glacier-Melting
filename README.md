# Black-Carbon-Impact-on-Himalayan-Glacier-Melting
Black carbon from fossil fuel/biomass combustion accelerates Himalayan glacier retreat via atmospheric heating &amp; surface darkening. AI analysis of 100K observations shows BC causes 13% of annual melting, peaks in March-April. Random Forest model (R²=0.895) identifies PM2.5 as key predictor. Threatens 750M people's water supply.
